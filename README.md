## Arsitektur

Struktur tingkat tinggi dari kode frontend dan disiplin untuk menciptakan struktur semacam itu.

+ **[Algoritma](architecture/algorithms.md)**: kemandirian langkah demi langkah dengan serangkaian operasi yang akan dilakukan. Algoritma melakukan perhitungan, pengolahan data, atau tugas penalaran otomatis.
+ **[Pola desain](architecture/design-patterns.md)**: Praktik terbaik yang bisa digunakan programmer untuk memecahkan masalah umum saat merancang aplikasi atau sistem.
+ **[Desain](architecture/designs.md)**: Siap digunakan dan didokumentasikan dengan baik struktur dan kerangka kerja untuk pengembangan frontend.
+ **[pemrograman Event-Driven (architecture/event-driven-programming.md)**: pemrograman Event-Driven adalah paradigma pemrograman dimana arus program ditentukan oleh kejadian seperti tindakan pengguna, output sensor, atau pesan dari program /urutan lain.
+ **[Pemrograman Fungsional](architecture/functional-programming.md)**: Pemrograman fungsional adalah paradigma pemrograman, yang memperlakukan perhitungan sebagai evaluasi fungsi matematis dan menghindari perubahan data dan mutable data.
+ **[Pemrograman Reaktif Fungsional atau Functional Reactive Programming (FRP)](architecture/functional-reactive-programming-frp.md)**: FRP adalah sebuah paradigma pemrograman untuk pemrograman dataflow asynchronous dengan menggunakan blok bangunan pemrograman fungsional.

## Kesesuaian

Kemampuan suatu produk untuk bekerja dengan perangkat input / output yang berbeda dan rendering perangkat lunak. Termasuk printer, e-mail, perangkat seluler dan browser yang berbeda.

+ **[Lintas browser](compatibility/cross-browser.md)**: Lintas browser atau Cross-browser mengacu pada kemampuan sebuah website, aplikasi web, membangun HTML atau script sisi klien berfungsi di lingkungan yang menyediakan fitur yang dibutuhkan dan untuk membungkuk atau menurunkan dengan anggun saat fitur tidak ada atau kurang.
+ **[E-Mail](compatibility/e-mail.md)**: Mempersiapkan e-mail berbasis HTML.
+ **[Keyboard](compatibility/keyboard.md)**: Bekerja dengan input keyboard di browser web.
+ **[perangkat seluler](compatibility/mobile.md)**: Pengembangan situs web yang dioptimalkan untuk dilihat di perangkat smartphone dan tablet.
+ **[Printers](compatibility/printers.md)**: Manipulasi output printer melalui CSS.
+ **[Desain Web yang Responsif atau Responsive Web Design (RWD)](compatibility/responsive-web-design-rwd.md)**: RWD menanggapi kebutuhan pengguna dan perangkat yang mereka gunakan. Perubahan tata letak berdasarkan ukuran dan kemampuan perangkat.
+ **[Aksesibilitas Web](compatibility/web-accessibility.md)**: Aksesibilitas web berarti orang-orang cacat dapat merasakannya, memahami, menavigasi, dan berinteraksi dengan Web, dan mereka dapat berkontribusi ke Web.

## Bahasa, Protokol, Browser APIs

Bahasa pemrograman / markup dan standar terkait web.

+ **[Cascading Gaya Lembar atau Cascading Style Sheets (CSS)](languages-protocols-browser-apis/cascading-style-sheets-css.md)**: CSS adalah bahasa stylesheet yang digunakan untuk menggambarkan presentasi dokumen yang ditulis dalam HTML atau XML. Ini menggambarkan bagaimana elemen harus diberikan pada layar, di atas kertas, dalam naskah, atau pada media lainnya.
+ **[Model Objek Dokumen atau Document Object Model (DOM)](languages-protocols-browser-apis/document-object-model-dom.md)**: DOM adalah antarmuka pemrograman untuk dokumen HTML, XML dan SVG. Ini mendefinisikan metode yang memungkinkan akses ke pohon, sehingga bisa mengubah struktur dokumen, gaya dan konten.
+ **[HyperText Markup Language (HTML)](languages-protocols-browser-apis/hypertext-markup-language-html.md)**: HTML adalah bahasa markup standar yang digunakan untuk membuat halaman web dan elemen-elemennya membentuk blok bangunan dari semua situs web.
+ **[Hypertext Transfer Protocol (HTTP)](languages-protocols-browser-apis/hypertext-transfer-protocol-http.md)**: Hypertext Transfer Protocol (HTTP) adalah protokol aplikasi untuk sistem informasi terdistribusi, kolaboratif, hypermedia. HTTP adalah dasar komunikasi data untuk World Wide Web.
+ **[JavaScript (EcmaScript)](languages-protocols-browser-apis/javascript-ecmascript.md)**: JavaScript adalah bahasa pemrograman dinamis penuh yang, bila diterapkan pada dokumen HTML, dapat memberikan interaktivitas dinamis di situs web. Hal ini didefinisikan dengan standar ECMAScript.
+ **[JavaScript Object Notation (JSON)](languages-protocols-browser-apis/javascript-object-notation-json.md)**: JSON (JavaScript Object Notation) adalah format data-interchange yang ringan. Mudah bagi manusia untuk membaca dan menulis. Mudah bagi mesin untuk mengurai dan menghasilkan. Hal ini didasarkan pada subset dari JavaScript Programming Language.
+ **[Scalable Vector Graphics (SVG)](languages-protocols-browser-apis/scalable-vector-graphics-svg.md)**: Format gambar vektor berbasis XML untuk grafis dua dimensi dengan dukungan untuk interaktivitas dan animasi.
+ **[Service Workers](languages-protocols-browser-apis/service-workers.md)**: Sebuah metode yang memungkinkan aplikasi memanfaatkan pemrosesan latar belakang yang gigih, termasuk kait untuk memungkinkan bootstrap aplikasi web saat offline.
+ **[Templating Languages and Engines](languages-protocols-browser-apis/templating-languages-and-engines.md)**: Mesin template adalah alat untuk memisahkan logika program dan presentasi menjadi dua bagian independen. Hal ini membuat pengembangan logika dan presentasi menjadi lebih mudah, meningkatkan fleksibilitas dan memudahkan modifikasi dan perawatan.
+ **[Transpiled Languages](languages-protocols-browser-apis/transpiled-languages.md)**: Bahasa abstrak diubah menjadi native, browser mendukung standar seperti JavaScript atau CSS.
+ **[Uniform Resource Identifier (URI)](languages-protocols-browser-apis/uniform-resource-identifier-uri.md)**: URI adalah serangkaian karakter yang digunakan untuk mengidentifikasi sumber daya. Bentuk URI yang paling umum adalah Uniform Resource Locator (URL).
+ **[Web Animations API](animation/web-animations-api.md)**: Web Animations adalah API JavaScript baru untuk menggerakkan konten animasi di web. Dengan menyatukan fitur animasi SVG dan CSS, Animasi Web membuka fitur yang sebelumnya hanya dapat digunakan secara terpisah, dan memperlihatkan kemampuan animasi berkinerja tinggi yang hebat kepada para pengembang.
+ **[WebAssembly](languages-protocols-browser-apis/webassembly.md)**: WebAssembly dimaksudkan untuk mengisi tempat yang telah dipaksa JavaScript untuk menempati sampai sekarang: representasi kode tingkat rendah yang dapat berfungsi sebagai target kompilator.

## Komponen antarmuka pengguna

Komponen UI bawaan untuk situs web dan aplikasi.

+ **[Tombol](user-interface-components/buttons.md)**: Tombol istilah mengacu pada elemen kontrol grafis yang memberi pengguna cara sederhana untuk memicu sebuah acara, seperti mencari kueri di mesin pencari, atau berinteraksi dengan kotak dialog, seperti mengonfirmasi tindakan.
+ **[kode](user-interface-components/code.md)**: Pemirsa kode dan editor dirancang untuk menyematkan di dalam situs web.
+ **[Formulir](user-interface-components/forms.md)**: Formulir HTML pada halaman web memungkinkan pengguna memasukkan data yang dikirim ke server untuk diproses. Pengguna web mengisi formulir menggunakan kotak centang, tombol radio, atau bidang teks.
+ **[Galeries dan Image Sliders](user-interface-components/galeries-and-image-sliders.md)**: Cara yang canggih untuk menyajikan koleksi gambar di situs Anda.
+ **[kisi](user-interface-components/grid.md)**: CSS Sistem Tata Letak kisi.
+ **[Editor kaya teks](user-interface-components/rich-text-editors.md)**: Editor teks kaya adalah antarmuka untuk mengedit teks kaya dalam browser web. Tujuannya adalah untuk mengurangi upaya pengguna yang mencoba mengekspresikan format mereka secara langsung sebagai markup HTML yang valid.
+ **[Daftar Isi](user-interface-components/table-of-contents.md)**: Komponen untuk pembuatan daftar isi otomatis.
+ **[UI Kits](user-interface-components/ui-kits.md)**: Koleksi komponen siap pakai.
+ **[Video dan Audio](user-interface-components/video-and-audio.md)**: Komponen untuk memutar file audio dan video di situs web.
